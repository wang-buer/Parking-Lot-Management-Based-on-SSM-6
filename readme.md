# 基于SSM的停车场管理6



## 1、项目介绍

基于SSM的停车场管理6拥有两种角色：管理员和用户

管理员：车位管理、分类管理、用户管理、入场停车管理、出场缴费管理、违规处罚管理、公告管理等

用户：登录注册、停车、缴费、投诉建议、收藏等
